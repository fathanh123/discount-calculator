# Discount Calculator

## Usage

Run the application:

```bash
python discount-calculator.py
```

### How to Use

1. Enter the original price in the "Original Price" field
2. Enter the discount percentage in the "Discount (%)" field
3. Click the "Calculate!" button
4. The final price will be displayed in Rupiah format

### Example

- Original Price: 100000
- Discount (%): 20
- Final Price: Rp 80,000.00
